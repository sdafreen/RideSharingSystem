# 🚖 Ride Sharing System

A Ride Sharing System developed using **Python** and **MySQL** that simulates the core functionalities of ride-hailing platforms like Uber and Ola. The project enables riders to book rides, drivers to manage ride requests, and administrators to oversee system operations.

## ✨ Features

- User & Driver Registration/Login
- Ride Booking & Cancellation
- Ride Acceptance & Completion
- Fare Calculation & Payment
- Ride History
- Driver KYC Verification
- Ratings & Feedback
- Admin Dashboard

## 🛠 Technologies Used

- Python
- MySQL
- SQL
- Object-Oriented Programming (OOP)
- Git & GitHub

## 🚀 How to Run

1. Clone the repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Import `database.sql` into MySQL.
4. Update your database credentials.
5. Run:
   ```bash
   python main.py
   ```
